# claude-plugins

Claude Code plugins by [allixsenos](https://github.com/allixsenos).

## Install

```
/plugin marketplace add allixsenos/claude-plugins
/plugin install <plugin-name>@allixsenos-claude-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [git-governor](plugins/git-governor/) | Enforce git governance — block amends, force pushes, commits to protected branches, and more |
